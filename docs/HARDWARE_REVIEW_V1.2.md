### 2.2 关键修复细节

#### B2：ASCII 引脚图（§2.1）修复详情

**修复前（V1.1）**：ASCII 图使用 WROOM-1 v1.3 旧版编号，Pin 22-27 显示为 IO46/IO9/IO10/IO11/IO12/IO13。

**修复后（V1.2）**：ASCII 图使用 WROOM-1 v1.8 Table 2-1 物理编号：
- Pin 22 = IO14 ✅（与 §2.2 一致）
- Pin 23 = IO21 ✅（与 §2.2 一致）
- Pin 24 = IO47 ✅（与 §2.2 一致）
- Pin 25 = IO48 ✅（与 §2.2 一致）
- Pin 26 = IO45 ✅（与 §2.2 一致）
- Pin 27 = IO0 ✅（与 §2.2 一致）

§2.1 注释明确说明编号来源，§2.2 表头标注参考数据手册版本，**彻底消除两处编号冲突**。

#### N1：ME6211C33 封装修复详情

**修复前（V1.1）**：§1.4 标注 SOT-89，但标了 4 信号（VIN/CE/VOUT/GND），与 SOT-89 实际 3 pin + tab 封装不符。

**修复后（V1.2）**：
- 封装更正为 **SOT-23-5**（5 引脚）
- 引脚定义明确：**VIN(1) / CE(2) / VOUT(3) / NC(4) / GND(5)**
- 完整型号 **ME6211C33M5G-N**（M5G = SOT-23-5，N = 无铅）
- 附录 A 选型表同步更新
- 修订记录 1.2 行明确标注 "SOT-89→SOT-23-5"

#### N2：泄放电阻修复详情

| 参数 | V1.1 | V1.2 | 改善 |
|------|------|------|:----:|
| 电阻值 | 1 kΩ | **100 kΩ** | 100× |
| 静态电流 | 3.3 mA | **33 µA** | 100× |
| 关断漏电流 | 不可接受 | 可接受 | ✅ |

100kΩ 配合 1µF 输出电容，泄放时间常数 τ = 100kΩ × 1µF ≈ 100ms，关断后 1s 内 VOUT 衰减至 < 1%，既保证快速关断又大幅降低静态功耗。

#### N3：VDD_SPI PCB 净空规则详情

§2.2 末尾新增注释（原文）：

> **VDD_SPI 域引脚 PCB 净空**：GPIO45/46/47/48 属于 VDD_SPI 电源域（驱动 Octal PSRAM），在 PCB Layout 中应确保上述引脚周边 **≥5mm 区域内无其他走线或元件**，避免 PSRAM 高速通信受数字噪声干扰。

这是 ESP32-S3 Octal PSRAM 应用的**关键设计规则**，正确添加。

## 三、V1.2 新发现/遗留问题（轻微级）

V1.2 实施过程非常扎实，未发现阻塞/重要问题，仅有 4 项轻微改进点：

### M4（轻微）：§2.2 GPIO 分配表 Pin 28-37 冗余

**现象**：§2.2 表中 Pin 28-37 重复列出物理 9/8/22/21/20/19/18/17/12（即 Pin 14-19 的镜像）。物理引脚 Pin 16（GPIO16）、Pin 15（GPIO15）等被列了 2 次。

**影响**：低。表内 "(见 Pin X)" 注释已说明，但会让 PCB 工程师初次阅读时困惑。

**建议**：将表第一列改为物理引脚号，去掉原理图位号列；或增加表头注释说明 "Pin 28-37 为镜像行，仅作原理图内部指引用"。

### M5（轻微）：§2.2 物理 Pin 1（GND）未列出

**现象**：§2.1 ASCII 图已列出物理 Pin 1 = GND，但 §2.2 分配表从物理 Pin 2 开始，物理 Pin 1（GND）未独立成行。

**影响**：低。GND 是公共连接，多数原理图不单独列。

**建议**：在表头添加一行 "Pin 1 (GND) — 模组公共地，详见 §2.1 原理图"，保持表格完整性。

### M6（轻微）：§6.3 P-MOSFET 隔离方案缺默认状态说明

**现象**：§6.3 改进方案使用 P-MOSFET (Si2301) 隔离电池分压网络，但未说明：
- GPIO_CTRL 默认电平（上电时是断开还是接通？）
- 测量时序（何时使能 MOSFET 测量？）

**影响**：低。需要固件配合，对硬件设计无影响，但文档不完整。

**建议**：在 §6.3 补充：
```
GPIO_CTRL 默认低电平（上电 = 断开 = 不测量）
固件测量电池电压时序：
  1. GPIO_CTRL 置高 → MOSFET 导通 → 分压网络连通
  2. ADC 读取 GPIO3 电压
  3. GPIO_CTRL 置低 → MOSFET 关断 → 降低漏电流
  4. 测量间隔 ≥ 1s，避免频繁切换
```

### M7（轻微）：§2.1 ASCII 图未标注 EPAD 引脚号

**现象**：§2.1 ASCII 图底部标注 "41: EPAD(GND)"，但未在 ASCII 图中明确位置（Pin 41 对应图中哪个位置）。

**影响**：极低。文字注释已说明。

**建议**：可在 ASCII 图中央底部 EPAD 处添加 "EPAD=41" 标识。
